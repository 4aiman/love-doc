love.image.newEncodedImageData (Русский)

Кодирует ImageData.

Функция

Вид

data = love.image.newEncodedImageData( imageData, format )

Аргументы

ImageData imageData
Исходные данные ImageData для кодирования.
ImageFormat format
Формат, в который надо кодировать изображение.

Возвращает

Data data
Кодированные данные изображения.
