love.draw (简体中文)

用来在每一帧渲染画面的回调函数.

函数

简介

love.draw( )

参数

无.

返回

无.

范例

绘制一个在love.load函数内被加载的图像(最好别把love.graphics.newImage放在love.draw里,这样会导致图像在每一帧都被加载一次).

function love.load()
   hamster = love.graphics.newImage("hamster.png")
   x = 50
   y = 50
end
function love.draw()
   love.graphics.draw(hamster, x, y)
end
