love.keypressed (한국어)

키를 눌렀을 때 작동하는 콜백 함수입니다.

함수

Available since LÖVE 0.9.0
This variant is not supported in earlier versions.

형식

love.keypressed( key, isrepeat )

매개변수

KeyConstant (한국어) key
눌린 키의 값.
boolean (한국어) isrepeat
사용자가 키를 계속 누르고 있는지 검사합니다. love.keypressed가 호출될 때의 딜레이는 각 시스템의 설정에 따라 다릅니다.

리턴값

없음.

알아 두기

사용자가 키를 계속 누르고 있는지 검사하려면 먼저 love.keyboard.setKeyRepeat를 활성화시켜야 합니다.
