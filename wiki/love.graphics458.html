love.graphics.setBackgroundColor (Русский)

Устанавливает цвет фона.

Функция

Вид

love.graphics.setBackgroundColor( r, g, b )

Аргументы

number r
Красная компонента (0-255).
number g
Зеленая компонента (0-255).
number b
Синяя компонента (0-255).

Возвращает

Ничего.

Функция

Available since LÖVE 0.7.0
This variant is not supported in earlier versions.

Вид

love.graphics.setBackgroundColor( rgb )

Аргументы

table rgb
Индексированная числовыми значениями таблица, содержащая красную, зеленую и синюю компоненты цвета.

Возвращает

Ничего.
