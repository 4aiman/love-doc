love (Português)

Módulo principal que contem todos os outros módulos.

Módulos

love.audio (Português) Provê uma interface para criar sons com os alto-falantes do usuário.
love.event (Português) Gerencia eventos, como por exemplo teclas pressionadas.
love.filesystem (Português) Provê uma interface para o sistema de arquivos do usuário.
love.font (Português) Permite trabalhar com fontes.
love.graphics (Português) Desenho de formas e imagens, administração da geometria da tela.
love.image (Português) Provê uma interface para decodificar imagens codificadas.
love.joystick (Português) Provê uma interface para o joystick do usuário.
love.keyboard (Português) Provê uma interface com o teclado do usuário.
love.mouse (Português) Provê uma interface com o mouse do usuário.
love.physics (Português) Pode simular a física da rigidez de corpos 2D de uma forma realista.
love.sound (Português) Este módulo é responsável por decodificar arquivos de som.
love.thread (Português) Permite trabalhar com threads.
love.timer (Português) Provê uma interface para o relógio do usuário.


Tipos

Data (Português) A superclasse de todos os dados.
Object (Português) A superclasse de todos os tipos do LÖVE.


Callbacks

love.draw (Português) Função de callback usada para desenhar na tela a cada quadro.
love.focus (Português) Função de callback executada sempre que a janela recebe ou perde o foco.
love.joystickpressed (Português) Chamada quando um botão no joystick é pressionado.
love.joystickreleased (Português) Chamada quando um botão do joystick é liberado.
love.keypressed (Português) Função de callback executada sempre que uma tecla é pressionada.
love.keyreleased (Português) Função de callback executada sempre que uma tecla é liberada.
love.load (Português) Esta função é chamada exatamente uma única vez no início do jogo.
love.mousepressed (Português) Função de callback executada sempre que um botão do mouse é pressionado.
love.mousereleased (Português) Função de callback executada sempre que um botão do mouse é liberado.
love.quit (Português) Função callback executada sempre que um jogo é fechado.
love.run (Português) A função principal, que contém o loop principal.
love.update (Português) Função de callback usada para atualizar o estado do jogo a cada quadro.
