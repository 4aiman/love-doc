love.mouse.getX (简体中文)

返回当前的鼠标x坐标。

函数

实例

x = love.mouse.getX( )

参数

没有.(没有写出来毛线啊)

返回

number x
鼠标沿x轴的位置

例程在这

在鼠标的x的位置画一条垂直线。

function love.draw()
   local x = love.mouse.getX()
   love.graphics.line(x,0, x,love.graphics.getHeight())
end
